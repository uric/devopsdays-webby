--- extension: html filter: - erb - markdown dirty: true proposal: false talk: false selected: false layout: event author: Michael Sverdlik title: "InfluxDB Workshop" --- **Abstract:** This workshop will introduce InfluxDB, a time series, events, and metrics database. It’s written in Go and has no external dependencies. That means once you install it there’s nothing else to manage (like Redis, HBase, or whatever). It’s designed to be distributed and scale horizontally, but be useful even if you’re only running it on a single box. **Speaker:** Michael Sverdlik, GigaSpaces.
